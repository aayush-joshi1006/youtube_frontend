## Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/aayush-joshi1006/youtube_frontend
cd frontend
```

### 2. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```

Frontend will run at: `http://localhost:5173/`
